EndoSystem (Demo)
MVP web mobile-first para registro de dor e apoio diário em saúde feminina (focado em endometriose). Esta é uma versão de demonstração técnica focada na interface e experiência do usuário (UI/UX).

🛠️ Tecnologias
HTML5 / CSS3 (Design Mobile-First)

JavaScript (ES6+)

localStorage (Persistência local de dados)

🚀 Funcionalidades Implementadas
Check-in Rápido: Registro de nível de dor (1-10) e botões de atalho para crises.

Seletor de Sintomas: Interface para validação de sintomas recorrentes.

Contexto Diário: Registro opcional de sono, estresse e gatilhos alimentares.

Acompanhamento de Ciclo: Registro manual das fases (menstrual, folicular, ovulatória e lútea).

Historial e Insights: Exibição de médias de dor, dias críticos e sintomas mais frequentes com dados persistidos localmente.

Modo Crise: Fluxo de suporte imediato com interface simplificada.
